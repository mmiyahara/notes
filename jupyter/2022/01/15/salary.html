うわっ・・・私の年収、低すぎ・・・？

転職サイトのネット広告として、あまりにも有名なフレーズです。

私は現在20代後半ですが、この広告を見たのは多分中学生か高校生の頃。
当時は「凄い煽り文句だな」程度にしか思わなかったですが、
社会人6年目の今、ここまで自分に刺さるフレーズは他にありません。

平均給与についてはネット記事などで散々擦られていることは重々承知ですが、
自分で調べてみないと納得・理解できない。
また、業種ごとや年齢ごとの平均給与についての記事は散見されますが、
転職などを検討するにあたって気になるのは「業種ごと、かつ年齢ごとの平均給与」です。

ということで、調べてみました。業種ごと、年齢ごとの平均給与については、
民間給与実態統計調査 第12表 業種別及び年齢階層別の給与所得者数・給与額で公開されています。

from dotenv import load_dotenv
import io
import os
import urllib
import requests
import pandas as pd
import seaborn as sns

# .env から環境変数を読み込む
load_dotenv()
ESTAT_APP_ID = os.getenv('ESTAT_APP_ID')
BASE_URL = 'https://api.e-stat.go.jp/rest/3.0/app/getSimpleStatsData'

# 産業別、年齢別の給与額
params_dict = {
    "appId": ESTAT_APP_ID,
    "statsDataId": "0003045948",
    "surveyYears": "2019",
    "explanationGetFlg": "N",
    "annotationGetFlg": "N",
    "metaGetFlg": "N",
    "sectionHeaderFlg": "2",
}
resp = requests.get(BASE_URL, urllib.parse.urlencode(params_dict))


df = pd.read_csv(io.StringIO(resp.text))

# 2019年に絞る
df = df[df["年"] == "2019年"]

# 平均給与のデータのみに絞る
df = df[df["表章項目"] == "平均給与"]

# 必要なカラムのみに絞る
df = df[["業種（2008年～）", "年齢", "value"]]

# 年齢 == "合計" (全年齢の平均給与)のデータを削除
df = df[df["年齢"] != "合計"]

# カラム名を変更
df = df.rename({"業種（2008年～）": "業種", "value": "平均給与"}, axis=1)

# 単位を 千円 から 万円 に変更
df["平均給与"] = df["平均給与"].astype(int) // 10

# 年齢を数値に変換
replace_dict = {
    "19歳以下": 19,
    "20～24歳": 24,
    "25～29歳": 29,
    "30～34歳": 34,
    "35～39歳": 39,
    "40～44歳": 44,
    "45～49歳": 49,
    "50～54歳": 54,
    "55～59歳": 59,
    "60～64歳": 64,
    "65～69歳": 69,
    "70歳以上": 70,
}
df["年齢"] = df["年齢"].replace(replace_dict)

df = df.reset_index(drop=True)


g = sns.catplot(x="平均給与", y="年齢", col="業種", data=df, kind="bar", aspect=.7, col_wrap=4, orient="h")
g.tight_layout()

fig = g.figure
fig.suptitle('2019年 産業別、年齢別 平均給与額(万円)', fontsize=16, y=1.015)
fig.text(x=0.03, y=-0.01, s='* 民間給与実態統計調査 民間給与実態統計 第12表 業種別及び年齢階層別の給与所得者数・給与額 より作成')
fig.text(x=0.03, y=-0.02, s='* 平均給与額(万円) 400万円に縦線を表示')

for ax in g.axes:
    # 各図に x, y ラベルを表示
    # ref: https://stackoverflow.com/questions/67390108/seaborn-facetgrid-how-to-show-y-tick-labels-in-all-subplots
    ax.tick_params(labelleft=True, labelbottom=True)
    # 平均給与400万円に縦線を表示
    ax.axvline(x=400, ymin=0, ymax=100, alpha=.3)
    for c in ax.containers:
        # 各棒グラフに平均給与を表示
        # ref: https://matplotlib.org/stable/api/_as_gen/matplotlib.axes.Axes.bar_label.html
        labels = [f"{v.get_width():.0f}" for v in c]
        ax.bar_label(c, labels=labels, label_type='edge', padding=2)

調査対象

民間給与実態統計調査 の調査計画に以下の記載があります。

源泉徴収義務者名簿を基に、源泉徴収義務者を国税局ブロック別、給与所得者数別に区分して無作為抽出する。

"源泉徴収義務者" には正社員はもちろん、派遣社員やアルバイトも含まれます。
正社員に限っての結果ではない点は注意が必要そうです。

グラフについて

  • 縦軸の値について、以下のルールで表示しています。
    • 19:19歳以下 - 24: 20歳以上24歳以下
    • 29: 25歳以上29歳以下
    • ...: ...
    • 69: 65歳以上69歳以下
    • 70: 70歳以上

感想

  • 業種によって、想像以上に差がありました。
  • 「卸売業、小売業」「サービス業」「宿泊業、飲食サービス業」の平均給与が
    他業種と比べて低いです。アルバイトや派遣社員が多い影響もあるのかもしれません。
  • 「情報通信業」の平均給与は他業種と比べて高いと言えそうです。中でも 59 (55歳以上59歳以下) の給与が高いです。
    現在50代後半ということは、インターネット黎明期(1990年代)から情報通信業を支えてきた方々。
    先行者利益と言っては語弊があるかもしれませんが、人数が少ないこともあり、ある程度の役職についていると想像できます。
    私は「情報通信業」に属する企業で勤務していますが、私が 50 代になる頃までこの平均給与が維持されているかは疑問です。
  • 「電気・ガス・熱供給・水道業」の平均給与が圧倒的に高いです。
    都道府県別電気・ガス・水道業界年収 によると、都道府県によっても給与に差があるとのこと。沖縄電力最強説。

総括

いかがでしたでしょうか。
これから社会人になるにあたり業界を検討している学生の方々や、
自身の給与に満足できず転職活動を検討している社会人の方々、
私と同じく「うわっ・・・私の年収、低すぎ・・・？」のフレーズが刺さる方々に、この結果が刺されば幸いです。
